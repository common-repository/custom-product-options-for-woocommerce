Looking for a free plugin to add extra fields and options to your WooCommerce products? This is the one you need!

== Description ==

= FREE WOOCOMMERCE PRODUCT ADD-ONS =

Customize your products with this easy to use plugin. Add your own fields like text fields, radio buttons, colors, dates and MANY MANY more to your products with ease and speed.

Want to add a field for custom engraving? Easy to do. Pick a color for a t-shirt - again super-easy. Capture an email address - no problem. The limit is only your imagination!

== WooCommerce Custom Fields Supported ==

* Text Field - a simple single line of text
* Text Area - for when you need to capture multiple lines of text (e.g. a greeting or message)
* Number - need to capture a number? We've got you covered
* Password - secret password time
* Email - validates you have a good email address
* Checkbox - For selecting options - want to put pepperoni on your pizza?
* Radio Button - When you want to pick one of several options
* File Upload - need your customer to provide a file - no problem
* Date Picker - let your customers pick their own date
* Color - Need your customer to pick a color? That was easy!
* Dropdown - Display a nice dropdown list to be picked from
* Paragraph - This allows you to include some text and other HTML elements on the page. typically used for instructions
* Header - H1, H2, H3, H4, H5, H6 - use again for formatting & instructions to the customer

== Decide which products these fields are added to ==

In addition to the fields the plugin provides you great flexibility on where to display these fields.

You can choose to display on certain products or categories. This gives you complete control of where you show your extra fields. Just on a single or few products, or all products in a category - you decide.

== Installation  ==

You can install this plugin similar to any other WordPress plugin.

1. Download and unzip the zip file.
2. Upload the entire plugin directory to your /wp-content/plugins/ directory.
3. Activate the plugin through the Plugins menu in WordPress Administration.

You can also use install straight from the plugins page on your WordPress site - search for "Custom Product Options for WooCommerce"
